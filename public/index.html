Const PostSchema

Inside this function, you can add any item you want for connect their to database after

In our case, we have 5 items :

  • A name (required)
  • A birthday
  • A weight (required)
  • A step
  • A status "isRunning"

CRUD

  • router.get('/')

    • Get all informations about all item(s) created


  • router.post('/')

    • Creat a new item with required value and an id


  • router.get('/:ChickenId)

    • Get all informations about an item selected by the id


  • router.delete('/:ChickenId)

    • Delete an item selected by the id


  • router.patch('/:ChickenId)

    • Modify a value from an item select by the id


  • router.patch('/run/:ChickenId)

    • Increase step value and change the status from "isRunning" to true


  • router.get('/run/:ChickenId)

    • Get all informations about all item(s) created and with step value increased and also isRunning status is true
